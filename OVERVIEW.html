🚀 Quick Setup Instructions

Prerequisites: Make sure you have Node.js installed on your system. Download from: https://nodejs.org/
  1. Open Command Prompt or PowerShell as Administrator
  2. Navigate to the project directory:
    cd C:\Users\sande\attendance-system
  3. Install dependencies:
    npm install
  4. Start the application:
    npm start
  5. Access the system:
    Open browser and go to: http://localhost:3000
  6. Login credentials:
    Admin: username = admin, password = admin123

🎯 Alternative Setup Methods:

  • Using batch files: Double-click install.bat then start.bat
  • Manual installation: See SETUP.md for detailed instructions
  • Troubleshooting: Check README.md for common issues and solutions
